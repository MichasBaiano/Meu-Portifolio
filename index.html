Meus Projetos

Brilho Perdido

Brilho Perdido é um RPG ambientado em fases misteriosas que exploram a cidade de Pedro II e suas lendas. A jornada leva o jogador por caminhos repletos de mitologia, segredos e desafios únicos, fundindo cultura local com elementos de fantasia e ação.

Capelão Bet

Aplicação web capelao.bet, que simula um sistema de apostas em eventos esportivos e no tradicional jogo do bicho. O projeto foi estruturado seguindo o padrão MVC, utilizando Python com o framework Flask e banco de dados SQLite.

Faz Frete

O Faz Frete é um aplicativo mobile planejado para otimizar o gerenciamento de fretes e o transporte de passageiros e mercadorias em Pedro II/PI e região.

Urna Eletrônica

Este projeto foi desenvolvido como trabalho final da disciplina de Programação Orientada a Objetos, com o objetivo de simular o funcionamento de uma urna eleitoral brasileira usando Java, tanto do ponto de vista visual quanto funcional.

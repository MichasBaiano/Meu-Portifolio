Capelao.BET

O Capelão.Bet é um sistema web desenvolvido com Flask(Python) para gerenciar apostas e eventos esportivos.

O projeto aplica o padrão MVC, com autenticação de usuários, administração de apostas e banco de dados SQLite, além de um frontend leve em HTML, CSS e JavaScript.

O resultado é uma aplicação organizada, intuitiva e escalável, que demonstra domínio prático de desenvolvimento web e arquitetura de software.

Funcionalidades Implementadas

Usuários

Cadastro e autenticação de usuários. Tela principal de navegação. Acesso a eventos disponíveis para aposta.

Apostas

Criação de apostas com base nos eventos. Armazenamento e listagem de apostas feitas.

Administração

Interface de administração com visualização de usuários, apostas e resultados. Cadastro e gerenciamento de eventos.

Eventos

Registro de novos eventos. Controle de eventos ativos e já finalizados.
